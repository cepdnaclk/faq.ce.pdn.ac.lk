--- layout: qa_page permalink: "/field-selection/prior-knowledge/" parent: "Field Selection" title: "Prior Knowledge" index: 2 author: "Dr. Mahanama Wickramasinghe and Dr. Isuru Nawinne" github: faq/field-selection/02_prior_knowledge.html --- Any prior knowledge required for the Computer Engineering field? No. We will teach you all. That is why we are here. Do we have to have any hardware knowledge at the beginning of the course? No. We will teach you all that you need during the course. What programming languages do we have to learn before coming to the department? You don't HAVE to learn anything. If you want to, learn a bit about C and C++. You will be learning several languages throughout the programme. I like to join the Computer Engineering degree programme. But I only have a little knowledge of programming. Is that going to make a difference? As long as you are up to the challenge, willing to work hard, it is not an issue at all. We have statistics to prove that a lot of very good Computer Engineering graduates did not have any programming knowledge before joining. Are there any courses that you will recommend us to do during these days so that we can improve some basic skills regarding the field? You don't need to learn anything specific. We teach everything needed. If you have spare time, you can try and do some extra programming work, or learn some basics about C, C++, Java, Assembly. You can also try some simple programming project, reach out to our lectures and they will be more than happy to guide you. Or else just use the spare time for some fun hobby.
