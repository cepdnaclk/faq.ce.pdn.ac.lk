This is a GPU server of the Department of Computer Engineering, University of Peradeniya.

You can use this server using the LDAP login for ce.pdn.ac.lk undergraduates (which is used for CO top floor lab, Aiken and Tesla) without sudo access.

How to connect to this server? Since ada.ce.pdn.ac.lk does not have a public IP, you can either SSH directly from inside the university network, or SSH through Tesla (which has a public IP) from outside the university network. Read how to SSH

Direct (from inside the university network):
ssh e14000@ada.ce.pdn.ac.lk
Jump through Tesla (from outside):
ssh -J e14000@tesla.ce.pdn.ac.lk e14000@ada.ce.pdn.ac.lk What is the IP address of this server? This server uses the internal IP address 10.40.18.12. This address is only valid within the Peradeniya network. What is the hostname of this server? ada.ce.pdn.ac.lk. This hostname is only valid within the Peradeniya network. How can I get access to this server? If you are an undergraduate of the Department of Computer Engineering, you already have access to this server. If not, email headce[at]eng.pdn.ac.lk. What is the hardware configuration of this server?
  • Intel(R) Xeon(R) w7-3565X CPU (32 cores, 64 threads)
  • 3 * NVIDIA RTX 6000 Ada Generation 48GB GPUs
  • 512 GB RAM
 What is the operating system of this server? Ubuntu 24.04.3 LTS How to install software on this server? First, try to install it using the conda package manager ( Read instructions ) .
Then, try singularity. Read more (the methodology is similar to Turing)
