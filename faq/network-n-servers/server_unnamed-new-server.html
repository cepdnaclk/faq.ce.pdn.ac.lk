This is a GPU server of the Department of Computer Engineering, University of Peradeniya.

You can use this server using the LDAP login for ce.pdn.ac.lk undergraduates (which is used for CO top floor lab, Aiken and Tesla) without sudo access.

Specifications
  • Gigabyte MW83-RP0
  • Xeon W7-3565X Processor
  • High Efficiency Air cooler for LGA-4677 socket
  • Kingston DDR5 64GB ECC RDIMM * 8 = 512GB RAM
  • Kingston KC3000 series 4TB M.2 SSD * 2
  • 10TB HDD * 2
  • VROC STD to RAID above drives
  • RTX6000 ADA 48GB * 3
  • Tower chassis with GPU support
  • 2000w Single SMPS
